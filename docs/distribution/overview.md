# GovCMS Distribution

The GovCMS Drupal distribution is a packaged version of Drupal which contains:

* a set of stable and popular modules (add-ons)
* a base Drupal theme for the public and an administration theme
* government focused tools and features

The Git repositories contain the main documentation and issue queues for [GovCMS](https://github.com/govCMS/govCMS).

There are more links on the [Finding resources](../index.md#finding-resources) page.
